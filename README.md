## Build Setup

```bash
# clone the project
git clone https://github.com/LuFaLiu/vue-element-data.git

# enter the project directory
cd vue-element-data

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:8089

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
